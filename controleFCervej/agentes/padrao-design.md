## Fonte

A fonte utilizada é a **Montserrat**.

## Cores principais

Paleta de cores principais para utilização no projeto, com opções para as tabelas, títulos, destaques e alertas.

| Hex | Amostra |
|-----|---------|
| `#1D1D2D` | ⬛ Preto azulado (fundo principal) |
| `#063852` | 🟦 Azul escuro |
| `#ACBBCD` | 🔵 Azul médio |
| `#A4A4A4` | ⬜ Cinza médio |
| `#EBEBEB` | ⬜ Cinza claro |
| `#FFC524` | 🟡 Amarelo destaque |
| `#BCDA97` | 🟢 Verde claro |
| `#FA0897` | 🟣 Rosa/magenta alerta |

## Ícones

Paleta de ícones utilizados pelas soluções AirBrain. As soluções AirBrain utilizam-se de paletas de ícones específicas e que são expandidas continuamente de acordo com a necessidade de implementação de novas ferramentas ou alterações de ferramentas existentes. Os ícones portam as cores primitivas discriminadas neste projeto e, em casos raros, cores extras.

### Ícones Mobile

> Conjunto de ícones otimizados para interfaces mobile, organizados em grade compacta sobre fundo cinza escuro (`#3A3A3A`). Incluem ícones de navegação, ações, categorias e status.

### Ícones Desktop

> Conjunto de ícones para interfaces desktop, com maior densidade e variações adicionais. Organizados em grade sobre fundo cinza escuro, incluindo ícones de ferramentas, mídias, ações e integrações.
